7m (SevenMinutes) : Ubiquitous Personal Radio

7m is an audio media server

that runs as a Mac Desktop App

iTunes

7m distributes audio contents managed by iTunes for ...

web browsers

supports Chrome/Firefox/Safari on Mac/PC

iPhone/iPad

Android smart phones / tablets

Demo 1

play a iTunes playlist in safari

Demo 2

and sync back played date at end

so that the list updates in real-time

Demo 3
